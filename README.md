# Revenue Advance EU Dashboard

This project is an interactive dashboard exploring EU funding programmes, including:

- Creative Europe  
- ESF+  
- Erasmus+  
- Horizon Europe  

## 🚀 Features
- Funding comparison chart  
- Overview of supported project types  
- Insights into EU collaboration requirements  

## 🛠️ Tech Stack
- Observable Framework  
- JavaScript  
- Node.js  

## ▶️ Run Locally
npm install  
npm run dev  

Then open: http://localhost:3000  

## 📊 Purpose
This project was developed as part of research on EU funding programmes related to digital innovation and sustainability (Digital Bauhaus context).
